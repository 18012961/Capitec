Loan Eligibility Simulator – Capitec Version

A full-stack loan eligibility and interest rate simulator built with:

- Spring Boot (Backend)**
- HTML, CSS, JavaScript (Frontend)**
- Real-time loan validation
- Dynamic product & purpose loading
- Interest rate calculation engine

This project provides an interactive UI that allows users to enter financial details, check loan eligibility, and calculate interest rates using predefined backend rules.


 Features

 Loan Eligibility Check  
- Validates age, income, employment duration, loan amount, and more  
- Returns detailed eligibility results  
- Displays validation errors in real time  

Interest Rate Calculator  
- Calculates estimated loan interest based on credit score, amount, and loan term  
- Uses backend business rules and scoring  

 Dynamic Product & Purpose Loading  
Backend returns a list of loan products → frontend populates the dropdown automatically.

Clean Modern UI  
The UI is styled based on a professional savings calculator design, featuring:
- Smooth spacing  
- Rounded cards  
- Two-column layout  
- Mobile-friendly structure
